man "valgc"
--summary--
compile target file with gcc and run it under valgrind

--syntax--
valgc [target]

--param--
:target -- name of the C file to compile and run (without .c is also accepted)

--options--
 -h, --help   -- show this help

--hint--
- If no target is given, all .c files in the current directory are compiled and executed.

enjoy :)
