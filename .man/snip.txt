man "snip"
--summary--
insert snippet which any function or sequences

--syntax--
snip [snippet] [line number] [target file] [options]

--param--
:snippet      -- function or sequences name
:line number  -- the line number where you going to insert the snippet
:target file  -- target file name

--options--
 -l, --list                    -- show available snippets
 -e, --edit [filename]         -- open snippet file with vim
 -add [targetpath] [filename]  -- create new snippet file
 -d, --delete                  -- Delete a specified snippet
 -f, --find                    -- Search for snippets by keyword
 -h, --help                    -- show this help

 -m, --multi [target file] [line number] [snippet1 snippet2 ...]
                               -- insert multiple snippets in order
    Insert multiple snippets into a single target file at once.
    You can specify as many snippet names as you like; they will
    be inserted sequentially at the specified position.

    Example:
    snip -m main 0 swap sort minmax
        → inserts swap.txt, sort.txt, minmax.txt at the top of main.c

    snip -m main 45 quicksort merge bubble
        → inserts quicksort.txt, merge.txt, bubble.txt after line 45

--hint--
- For insertion, specify the snippet name, line number, and target file.
- When adding, provide a source file and new snippet name.
- Listing shows all snippets in the snippets folder.
- Editing opens the snippet in vim.
- Deletion removes the snippet permanently.
- Searching lists snippets whose filename or content matches the keyword.
- Use '-m' when you want to insert multiple snippets at once.
  For example: snip -m main 0 utils_init utils_free

enjoy :)
